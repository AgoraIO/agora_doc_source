## v4.2.0 版

该版本于 2023 年 5 月 23 日发布，是 C# SDK 的第一个版本。

#### 功能简介

该版本基于 RTC Windows SDK v4.2.0 封装，功能特性详见 [Windows 发版说明](./release_windows_ng?platform=Windows)。
