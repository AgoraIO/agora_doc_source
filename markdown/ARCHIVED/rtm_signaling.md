实时消息系统将逐步替代信令。除了高可靠、高并发、全球化部署等优势之外，Agora RTM SDK 将会持续更新，不断推出更多更丰富的功能和场景应用。

> - Agora RTM SDK  v1.0 支持与 Agora Signaling SDK 互通。
> - 对已有 Agora Signaling SDK 的维护将于 2019 年第四季度停止。
