# Extion App Context

## AgoraExtAppProtocol

### launchExtApp

```kotlin
- (NSInteger)willLaunchExtApp:(NSString *)appIdentifier;
```

| 参数            | 描述            |

### getRegisteredExtApps

```kotlin
fun getRegisteredExtApps(): List<AgoraExtAppInfo>
```
