本页提供 Agora RTSA Pro SDK 的发版说明。

## 2.1.0 版

该版本于 2020 年 12 月 xx 日发布。这是 Agora RTSA Pro SDK 的首个版本，包含以下功能：

- 发送和接收多种格式的音视频数据。
- 与 RTC Native SDK 和 RTC Web SDK 的双向互通。
- 支持灵活开始和暂停音视频流的发送和接收，支持监听远端音视频流的传输状态。
- 支持 Data Stream。
- 支持使用 String 型 UID 加入频道。
- 支持云代理服务。
- 支持上行带宽预测和关键帧等弱网对抗策略。
