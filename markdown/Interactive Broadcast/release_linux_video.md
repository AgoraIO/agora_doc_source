本文提供 Agora 完整包的发版说明。

## **简介**

Linux 完整包支持两种主要场景:

-   音视频通话

-   音视频直播


## 2.2.0 Beta 版

该版本于 2018 年 3 月 21 日发布。新增特性与修复问题列表详见下文。

Linux Client SDK 首版上线，支持如下功能：

-   通信场景下，音视频双向通信

-   视频设备信息收集

-   视频设备管理


**适用硬件类型**

-   x86 架构

-   Centos/Ubuntu 64 操作系统
