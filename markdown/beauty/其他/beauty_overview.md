实时音视频互动中，美颜功能可以帮助用户呈现更好的精神面貌，提供丰富有趣的贴纸特效。声网与第三方厂商合作，开发声网场景化 API 让开发者能在实时音视频互动中低门槛集成美颜功能。声网场景化 API，又称 Beauty API，封装了声网 RTC SDK 和第三方美颜 SDK 中的 API 调用逻辑，让开发者能够仅凭几行代码就能在声网的实时音视频互动逻辑中开启第三方厂商提供的丰富的美颜功能。
