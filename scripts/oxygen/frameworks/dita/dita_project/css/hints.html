Hints for DITA-OT Project elements

Main elements

project

The DITA Project file describes all the deliverables that will be contained from your DITA content.

include

Use this element to point to another project file, including its content into your current project.

context

A context defines a logical deliverable, it specifies a root map and optionally a number of DITAVAL filters.

publication

Define the DITA-OT transformation type and parameters for that transformation.

deliverable

Defines a specific deliverable, linking a logical deliverable defined by a context and a publication. Additional information like teh output location is also specified for a deliverable.
